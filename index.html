Interjektionen aus der DWDS-Lemmadatenbank

Einführung

Diese Webseite präsentiert eine umfassende Liste von Interjektionen, die im Digitalen Wörterbuch der deutschen Sprache (DWDS) verzeichnet sind. Interjektionen, auch Ausrufe oder Empfindungswörter genannt, sind Wörter, die oft zur Ausdrucksverstärkung verwendet werden und nicht in die grammatikalische Struktur des Satzes eingebunden sind. Beispiele sind “ach”, “oh”, “aha”, die emotionale Reaktionen, Überraschung oder andere spontane Empfindungen ausdrücken.

Erstellung der Liste

Die Erstellung der Liste erfolgte in mehreren sorgfältig ausgeführten Schritten, die im Folgenden beschrieben werden:

  1. Download der Daten von der DWDS-Lemmadatenbank:
    • Der erste Schritt bestand darin, die vollständige Datenbank der Wörterbucheinträge von der DWDS-Website herunterzuladen. Dies geschah über die Seite DWDS Lemmaliste, wo die CSV-Datei am Ende unter “Download” verfügbar ist.
  2. Auswahl der relevanten Informationen mit SQL:

    • Nachdem die CSV-Datei heruntergeladen wurde, wurde sie in Visual Studio Code geöffnet. Mit der Extension “Rainbow CSV” konnte die Tabelle übersichtlich dargestellt und bearbeitet werden. Durch eine gezielte SQL-Abfrage wurden nur die Zeilen mit der Wortart “Interjektion” ausgewählt:

      SELECT * WHERE a.Wortklasse == "Interjektion"
  3. Formatieren der Information mit Regex:
    • Um die ausgewählten Daten in ein geeignetes Format für die Webseite zu bringen, wurden reguläre Ausdrücke (Regex) verwendet. Die Einträge wurden so formatiert, dass sie den Markdown-Syntax für Listen und Links entsprechen.
  4. Schreiben des Begleittexts in Markdown:
    • Ein erläuternder Text wurde in Markdown geschrieben, um die Liste zu erklären und den Erstellungsprozess zu beschreiben.
  5. Umwandeln in HTML:
    • Der in Markdown erstellte Text und die Liste wurden mit dem Tool Pandoc in HTML umgewandelt. Hierbei wurde die Option “standalone” verwendet, um eine ansprechende und vollständige Webseite zu generieren.
  6. Hochladen auf Github und Veröffentlichung:
    • Abschließend wurde die HTML-Datei auf Github hochgeladen und veröffentlicht. Der Link zur veröffentlichten Webseite dient als Endprodukt dieser Aufgabe.

Die folgende Liste ist als direkter Link zum entsprechenden Eintrag im DWDS-Wörterbuch versehen, um weitere Informationen zu bieten.
